# project-28
### client and server are different projects

## to clone this repo 
<code> git clone git@github.com:Sanket-Gawande/project-28.git </code>

## to start client
<code> cd client </code>
<br>
<code> yarn && yarn dev </code>


## to start server
<code> cd server </code>
<br>
<code> yarn && yarn dev </code>

###### if you are using npm use "npm i && npm run" dev instead
